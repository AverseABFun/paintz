#PaintZ

PaintZ is a simple drawing program for the web.

##Live instance

The official live instance of PaintZ can be found at https://paintz.zmyaro.com.

Updates are posted to [Facebook](https://www.facebook.com/PaintZApp), [Google+](https://plus.google.com/+PaintZApp), and [Twitter](https://twitter.com/PaintZApp).
